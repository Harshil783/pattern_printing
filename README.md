# Patterns

Project on Patterns

## Installation

Download the source files and compile it.

### Linux
```bash
g++ main.cpp -o patterns.out

./patterns.out
```
### Windows
```cmd
g++ main.cpp -o patterns.exe

patterns.exe
```
